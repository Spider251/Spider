4. Linux常用命令
    1. ifconfig : 查看IP地址和MAC地址
        # windows : ipconfig
    2. ping IP地址/域名 -c 2
    3. nslookup : 解析域名对应的IP地址
        nslookup www.baidu.com
    4. top : Linux下任务管理器, 动态显示当前进程CPU及内存使用率, q退出
        top
        按 shift - > 或者 < 进行翻页
        按 q 退出
    5. ps -aux : 显示进程号(PID)
       ps -aux | grep "mysql"
    6. kill : 杀死1个进程
       sudo kill [-9] PID号
    7. df -h : 查看磁盘使用情况
    8. ls -lh 文件名
    9. chmod : 修改文件权限
        查看权限 : ls -l
        chmod +x test.py
        chmod 644 文件名
        rw-r--r-- : r(4) w(2) x(1)
    10. wc -l : 统计文件的行数
        wc -l /etc/passwd : Linux下用户数
    11. sort : 对文件中的内容进行排序
        sort 文件名
    12. uniq -c
        去除重复行,并统计每行出现的次数
        sort 文件名 | uniq -c
        
    13. ifconfig | head -2 | tail -1
        显示ifcofig的第二行的数据
    14. ssh : 远程连接到服务区
        格式 : ssh 用户名@IP地址
                ssh tarena@172.40.x.x
    15. scp
        远程复制文件/目录
        scp 文件名 用户名@IP地址
